Charles - это промежуточный сервер между клиентом и конечным сервером. Позволяет просматривать весь трафик HTTP и SSL / HTTPS.

Анализ запросов и ответов: статус код, метод, хост к которому обращаемся, путь запроса, старт запроса, время обработки запроса, размер получаемой информации.

Map Remote -подмена хостов (переадресация запросов с одного хоста на другой)
Breakpoint -перехват запроса и ответа
Rewrite -перехват запроса (можем менять данные, которые должны прийти в ответе)
Throttling
Block List -симуляция ошибок (статус код)
Allow List -исключает хост, действует на все другие.
Map Local-подмена ответа сервера в целом (можно подменить фоткой)
